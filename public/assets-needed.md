## Image Specifications

### Technical Requirements:
- **Format**: JPG for photos, PNG for logos with transparency
- **Quality**: High resolution, web-optimized
- **Aspect Ratios**: 
  - Hero images: 16:9
  - Service images: 4:3
  - Team photos: 1:1 (square)
  - Logos: Maintain original proportions

### Style Guidelines:
- **Professional appearance**: Clean, modern, trustworthy
- **Color consistency**: Should work with red (#8B1538) brand color
- **Lighting**: Well-lit, clear, professional photography
- **Composition**: Clear subjects, minimal distractions

## How to Add Images

1. **Save images** in the `public/` folder with descriptive names
2. **Update asset paths** in `src/data/assets.ts`
3. **Test images** across different screen sizes
4. **Optimize file sizes** for web performance
